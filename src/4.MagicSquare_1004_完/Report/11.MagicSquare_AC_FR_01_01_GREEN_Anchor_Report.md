## 6. CLI 수동 검증 (PYTHONPATH)

### 증상

프로젝트 루트에서 bare `python -c "from boundary..."` 실행 시:

```text
ModuleNotFoundError: No module named 'boundary'
```

### 원인

| 실행 방식 | `src/` 경로 |
|-----------|-------------|
| `pytest` | `pyproject.toml` `pythonpath = ["src"]` + `tests/conftest.py` |
| bare `python -c` | **자동 추가 없음** |

### 해결 (Git Bash)

```bash
PYTHONPATH=src python -c "
from boundary.input_validator import InputValidator
r = InputValidator().validate(None)
print(r.model_dump())
assert r.type == 'ERROR'
assert r.error.code == 'INVALID_SIZE'
assert r.error.message == 'Grid must be 4x4.'
print('OK')
"
```

**기대 출력:**

```json
{'type': 'ERROR', 'error': {'code': 'INVALID_SIZE', 'message': 'Grid must be 4x4.'}}
OK
```
