## 10. 알려진 정합 이슈

### ISS-012-01 — G1 Step A vs Step B (Report/02·PRD vs 수학)

| 관점 | G1 결과 |
|------|---------|
| PRD AC-FR05-01 / Control / U-OUT-01 / UI (G1) | `[2, 2, 7, 3, 3, 10]` Step A 라벨 |
| `TwoCellSolver` physical (Step B restores G0) | `[2, 2, 10, 3, 3, 7]` |
| 워크북 변형 격자 (예: (2,4),(4,1) 빈칸) | Solver `[2, 4, 8, 4, 1, 4]` vs Control `[2, 4, 4, 4, 1, 8]` |

**권장 후속:** `defect_list.md` 등록 → Control가 `SolutionResult.values` 반환 통일 REFACTOR 또는 G1 fixture 재정의.

### ISS-012-02 — 워크북 격자 vs G1

앱 **기본값 = G1**. spinbox 수정 시 결과는 pytest G1 고정값과 달라질 수 있음 — **정상** (다른 퍼즐 입력).
