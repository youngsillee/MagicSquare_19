```bash

python -m pytest tests/entity/ tests/control/ \
  --cov=src/entity --cov=src/control \
  --cov-report=term-missing

  --cov=src/boundary \
```

### 6.1 D-SOL-01 / G1 Step B 참고

`test_d_sol_01_g1_step_a_returns_solution`은 G1에서 Step A(smaller→first)가 실패하고 Step B(larger→first)로 해를 찾는다. 기대값 `[2,2,10,3,3,7]`. 따라서 L65(I8 Step A 성공)는 D-SOL-01만으로는 커버되지 않으며, Step A 전용 격자 또는 D-SOL-02(G2)가 필요하다.
