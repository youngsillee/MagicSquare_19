# Prompt 작업 보고서

1. 보고서를 내보내기
2. 프롬프트를 저장하기

## 수행 내역
- 기존 보고서/프롬프트 파일 네이밍 규칙 확인
- 현재 세션 기준 신규 보고서 파일 생성
- 현재 세션 대화를 대화형 프롬프트 transcript로 정리
- Prompt 폴더에 transcript 저장
